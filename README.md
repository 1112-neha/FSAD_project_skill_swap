# 🎯 FSAD Skill Swap Project

## 📌 Project Title

**Skill Swap Platform**

## 📖 Description

The Skill Swap Platform is a web application where users can exchange skills with each other.
Users can offer skills they know and request skills they want to learn, enabling peer-to-peer learning.

---

## 🚀 Features

* 👤 User Registration & Login
* 🧑‍🏫 Add / Manage Skills
* 🔄 Skill Exchange Requests
* 📩 Accept / Reject Requests
* 📊 Dashboard for tracking swaps
* 🔍 Search users by skills

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/FSAD_project_skill_swap.git
cd FSAD_project_skill_swap
```

### 2️⃣ Backend Setup (Spring Boot)

```bash
mvn clean install
mvn spring-boot:run
```

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## 🔗 API Endpoints (Example)

* `POST /api/users/register` → Register user
* `POST /api/users/login` → Login
* `GET /api/skills` → Get skills
* `POST /api/swap` → Request swap
